Песочница Python

Здесь можно писать команды на языке Python, и они будут моментально выполняться (по нажатию на клавишу ВВОД).

Для того, чтобы создать обычный файл с программой, выберете пункт Новая программа из меню Программа.

Для того, чтобы перезапустить интерпретатор в песочнице и забыть про все ранее выполненные строки, введите команду reset()
